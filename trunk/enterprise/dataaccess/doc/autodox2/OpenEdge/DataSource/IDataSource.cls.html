SetDataTarget

PUBLIC VOID SetDataTarget(HANDLE phData)
This SetDataTarget override is provided since the bulk of datatargets in ABL

will be buffers (possibly datasets). An exception will be thrown if the handle has a type of anything other than 'buffer' or 'dataset'.
Parameters:
handle - A buffer or dataset handle that is the data target.

SetDataTarget

PUBLIC VOID SetDataTarget(LONGCHAR pcData, DataFormatEnum poDataTargetFormat, DataTypeEnum poDataTargetType)
The SetDataTarget() method is called by the DataAccess object, to specify

the target for the DataSource's operations (usually the fill/fetch). This data is formatted per the MessageDataFormat above; and is stored in the message as either LONGCHAR, MEMPTR, HANDLE or P.L.O. The method performs the conversion from longchar to one of the other types, using ABL methods like COPY-LOB or WIDGET-HANDLE() or the OpenEdge.Lang.WeakReference object.
Parameters:
lonchar - Stores a handle, values or memptr contents
DataFormatEnum - The format of the data target value
DataTypeEnum - The data type of the value, for conversion.
