ResolveFieldName

PUBLIC CHARACTER ResolveFieldName(CHARACTER pcSourceTable, CHARACTER pcSourceField)
Resolves a field name from the Business Entity into a physical field name in the

DataSource. The values passed in here are usually taken from the ITableRequest that's part of the IServiceRequest.
Parameters:
character - A field name in the business entity
Returns:
character[] The [db.][table.]field name in the datasource. If the array has an extentof 1, then it's a field in the input table; if 2 then in the same Db. [1] = FieldName [2] = TableName [3] = DB Name

ResolveFilter

PUBLIC QueryFilter ResolveFilter(QueryFilter poSourceFilter)
Resolves a query filter on the data access/entity ("eTable.Field = 'someValue' ") to a filter on the

DB/datasource ("Table.Field = 'someValue'). This is typically for more complex transformations than just a simple field and/or table name mappes: for example, where one filter in the entity maps to multiple filters in the DB.
Parameters:
QueryFilter - The source/business entity filter to transform.
Returns:
QueryFilter[] The filter clause(s) on the DB query that the source filter resolves to

ResolveJoin

PUBLIC QueryJoin ResolveJoin(QueryJoin poSourceJoin)
Resolves a query Join on the data access/entity ("eTable.Field = eTable2.Field") to a Join on the

DB/datasource ("Table.Field = dbTable2.Field"). This is typically for more complex transformations than just a simple field and/or table name mappes: for example, where one Join in the entity maps to multiple Joins in the DB.
Parameters:
QueryJoin - The source/business entity Join to transform.
Returns:
QueryJoin[] The Join clause(s) on the DB query that the source Join resolves to

ResolveSort

PUBLIC QuerySort ResolveSort(QuerySort poSourceSort)
Resolves a query sort on the data access/entity ("by eTable.Field ") to a sort on the

DB/datasource ("By Table.Field"). This is typically for more complex transformations than just a simple field and/or table name mappings
Parameters:
QuerySort - The source/business entity sort to transform.
Returns:
QuerySort[] The sort clause(s) on the DB query that the source sort resolves to
