PUBLIC VOID GetDataTarget(OUTPUT LONGCHAR pmData, OUTPUT DataFormatEnum poMessageDataFormat)
The GetDataTarget() methods can be called by the DataSource object, to specify

the target for the DataSource's operations (usually the fill/fetch). This data is formatted per the MessageDataFormat above; and is stored in the message as either LONGCHAR, MEMPTR, HANDLE or P.L.O.
