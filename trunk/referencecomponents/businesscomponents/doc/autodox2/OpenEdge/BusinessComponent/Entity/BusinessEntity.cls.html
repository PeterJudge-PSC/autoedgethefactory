ExecuteRequest

PUBLIC VOID ExecuteRequest(IServiceRequest poRequest)
Services a request. The service or adapter will call ExecuteResponse() in the ServiceMessageManager

once it's done with the request. The object that implements this interface will call back to the ServiceMessageManager when it is done and it's ready with a response (IServiceResponse).
Parameters:
poRequest - The request to service.
poServiceMessageManager - The message manager controlling this request.
