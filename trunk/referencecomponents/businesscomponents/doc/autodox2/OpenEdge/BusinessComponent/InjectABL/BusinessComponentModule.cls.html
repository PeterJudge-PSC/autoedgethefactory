BindWorkStep

PROTECTED VOID BindWorkStep(CHARACTER pcBindingName, CHARACTER pcConcreteType, CHARACTER pcDecoratedType)
Workstep binding to service interface, workstep

A Workstep can be decorated as BizLogic workstep. In that case, we IWorkStep -> SomeBizLogicWorkStep decorates SomeWorkStep If not, then the we use bormal binding IWorkStep -> SomeWorkStep This possible decorated workstep now also needs to be exposed as a service interface WorkStepServiceInterfaceType decorates IWorkStep (whih is either SomeWorkStep or SomeBizLogicWorkStep)
