PROTECTED IMap GroupRequestsByService(IServiceRequest poRequests, LOGICAL plSyncRequest)
Groups a set of requests by service. There's no sorting done on the service; the grouped

requests are created in the order in which they are found in the passed message collection.
Parameters:
IServiceRequest - An array of messages (requests) which are being grouped.
logical - Whether the request is synchronous or not. If not so, maintain amap of the request IDs and the consumers.
Returns:
IMap A collection of service requests, keyed per service.
