ExecuteRequest

PUBLIC VOID ExecuteRequest(IMessageConsumer poMessageConsumer, IServiceRequest poRequests)
Executes the request(s) asynchroronously. Services a request. The service

provider will call ExecuteResponse() in the ServiceMessageManager once it's done with the request and ready with a response (IServiceResponse).
Parameters:
IMessageConsumer - The object initiating this request that will bereceive the responses as they are received by the Service Message Manager.
IServiceRequest - An array of the request(s) to be executed.
