Property Detail

AllowExternalJoins

LOGICAL PUBLIC GET SET
Indicates whether the query can join to tables outside of this query definition. This may

happen in some cases; for instance when dealing with DataSourceQuery objects which are used for DATA-SOURCE queries for ProDataSets. An external join is assumed to apply only to the 2nd (JoinBuffer) buffer in the QueryJoin.
