Method Detail

ExecuteRequest

PUBLIC VOID ExecuteRequest(IServiceRequest poRequest)
Services a request. The service provider will call ExecuteResponse() in

the ServiceMessageManager once it's done with the request and ready with a response (IServiceResponse); typically this will happen in a callback of some sort.
Parameters:
IServiceRequest - An array of requests to service.

ExecuteSyncRequest

PUBLIC IServiceResponse ExecuteSyncRequest(IServiceRequest poRequest)
Services a request in a synchronous manner. The responses are returned from the

ExecuteSyncRequest() call, rather than the service provider calling ExecuteResponse() in the ServiceMessageManager at a later time.
Parameters:
IServiceRequest - An array of requests to service.
Returns:
IServiceResponse An array of responses to the request.
