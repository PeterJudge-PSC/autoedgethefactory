Mapping from ABL data type to XML Schema supported data types. Taken from

the Working With XML book from the documentation set. Note that the converse is not supported, since there are multiple ABL types that map to a single XML schema type.
Parameters:
DataTypeEnum - The ABL data type
Returns:
character The XML data type.
